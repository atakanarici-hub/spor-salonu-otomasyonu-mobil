# 📱 Spor Salonu Mobil Otomasyonu (Android - Java)

Bu uygulama, spor salonu üyelerini yönetmek için geliştirilmiş, modern mimari prensiplerine dayalı bir Android uygulamasıdır. Masaüstü uygulaması ile tutarlı bir deneyim sunar.

## 🛠 Teknik Mimari
* **Dil:** Java
* **Veritabanı:** Room Database (SQLite Wrapper)
* **UI Komponentleri:** RecyclerView, CardView, Spinner
* **Mimari Yapı:** Singleton Database Instance, DAO Pattern, Entity-Relationship

## ✨ Özellikler
* **Güvenli Giriş:** Kullanıcı adı ve şifre kontrolü ile erişim.
* **Modern Liste:** `RecyclerView` ile yüksek performanslı üye listeleme.
* **Tam CRUD:** Üye ekleme, silme ve dinamik güncelleme.
* **Hızlı Erişim:** Liste öğelerine tıklayarak verileri otomatik forma taşıma.

## 🚀 Kurulum
1. Projeyi klonlayın.
2. Android Studio ile açın ve Gradle senkronizasyonunun bitmesini bekleyin.
3. Herhangi bir emülatör veya gerçek cihazda çalıştırın.
